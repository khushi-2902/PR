# AI-Powered Indian Heritage Learning Platform

## 🎯 Objective
To build an AI-integrated educational platform that provides structured information about Indian monuments and dynamically adapts explanations based on user learning modes.

---

## 🏗 Tech Stack

Backend:
- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT Authentication
- Bcrypt

Frontend:
- React.js

AI Integration:
- OpenAI / Gemini API

---

## 📌 Core Features

### 1. Authentication
- User Registration
- User Login
- JWT-based Authorization
- Protected Routes

### 2. Monument Management
- Add Monument
- View All Monuments
- View Single Monument
- Update Monument
- Delete Monument
- Search & Filter

### 3. AI Features
- Student Mode Explanation
- Exam Mode Explanation
- Tourist Mode Explanation
- Ask AI Custom Question
- Auto Quiz Generation
